# Pingr
## About:
**Pingr allows your application to save any logs it encounters during production and access them.**

Apart from trivial functions such as filtering logs, it also smartly notifies you with emails.
Critical errors are, well, critical. That's why Pingr will immediately send you an email if you get any critical errors.
For other non-critical information it will send you an AI-generated summary of what logs got added since the past 30 minutes.
## Quickstart guide:
### Requirements:
- Docker
- Ollama with Qwen3:8b installed locally
